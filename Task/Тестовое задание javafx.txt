Тестовое задание для проверки навыков работы с JavaFX.

Нужно реализовать простое приложение для создания заметок.
Приложение состоит главного экрана и нескольких диалоговых окон.

Главное окно состоит из 3-х частей: 
1. Панель работы с заметками. Включающая в себя дейтсивя: создание, редактирование и удаление заметки.
2. Список созданных заметок
3. Окно просмотра выделенной в списке заметки.

Каждая "заметка" состоит из Заголовка, Времени создания и Текста.
Заголовок и Текст могут быть отредактированы. Время создания - нет.

При нажатии кнопки "+" (создание) открывается диалог создания новой заметки ("редактор заметок", параметр: создание).
При нажатии кнопки "-" (удаление) открывается запрос на удаление заметки. В Диалоге нужно выводить Заголовок заметки.
При нажатии кнопки "..." (редактирование) должен открыться диалог редактирован (тот же что "редактор заметок", но обработан параметр "редкатирование").

Особенности реализации.
1. НЕ нужно заботится о "сохранении" заметок в файл или куда-то ещё. Все заметки находятся в оператирвной памяти и исчезают при закрытии приложения. Но если будет реализовано сохранение это будет дополнительным плюсом.
2. Окна (главное и редактор) должны быть созданы через FXML и иметь свои контроллеры. При растягивании окна интерфейс должен масштабироваться вмесье с окном.
3. Должно быть не мении 4х классов: Модель заметки, Приложение, Контроллер главного экрана, Контроллер редактора.
4. Проект должен основываться на maven - архетипе.
5. Результат работы - проект с исходными кодами и jar-архив с приложением, который можно запустить для проверки работы приложения. Проект должен быть опубликован на github-е или другом репозитории, допускаеися но не рекомендуется передача проекта как архива.
6. Диалект язка: java 8.
7. При релизации необходимо предусмотреть все нужные проверки (удаление из пустого списка, редактирование не выделенного элемента,...).
